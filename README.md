# SmartCommerce - E-Commerce Product Management System

## Overview
SmartCommerce is a modern E-Commerce Product Management System designed to manage products, inventory, orders, customers, and analytics through an intuitive dashboard and customer-facing shopping interface.

## Features

### Customer Features
- User Authentication
- Product Browsing
- Product Search & Filtering
- Shopping Cart
- Wishlist
- Order Tracking
- Product Reviews & Ratings
- Profile Management

### Admin Features
- Product Management
- Category Management
- Inventory Tracking
- Order Management
- Customer Management
- Sales Analytics
- Dashboard Reports

## Technology Stack

### Frontend
- React
- TypeScript
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express.js

### AI Integration
- Google Gemini API

## Installation

1. Clone the repository

```bash
git clone https://github.com/sharanya22-2005/Ecommerce-product-management-system.git
```

2. Install dependencies

```bash
npm install
```

3. Create `.env.local`

```env
GEMINI_API_KEY=your_api_key
```

4. Run the application

```bash
npm run dev
```
